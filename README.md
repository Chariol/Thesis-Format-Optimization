# BUPT 学位论文格式化工具

自动格式化北京邮电大学硕/博士学位论文（`.docx`），严格遵循《模板-理工科（2025年1月15日）》中全部 39 条批注规范。

## 功能

- **正文格式**：一至四级标题、正文、摘要、关键词、目录、图表题注、公式、脚注、参考文献、致谢等段落的字体/字号/行距/缩进/大纲级别
- **公式自动编号**：分章编号 `(X-Y)`，右对齐，居中显示
- **页眉页脚**：奇数页"本章标题"（STYLEREF 域）、偶数页"北京邮电大学硕士学位论文"；前置部分罗马页码、正文阿拉伯页码
- **页面设置**：A4、上下 2.54cm、左右 3.17cm
- **空行清理**：删除表格/图片前后多余空行
- **变更报告**：输出每个段落的格式修改明细（字体/字号/加粗/对齐）
- **图表编号检查**：验证图、表分章编号是否连续正确

## 快速开始

```bash
# 1. 安装依赖
pip3 install lxml

# 2. 克隆仓库
git clone https://github.com/Chariol/-.git
cd -

# 3. 格式化你的论文
bash format.sh 你的论文.docx
```

## 文件说明

| 文件 | 作用 |
|------|------|
| `format.sh` | 入口脚本，串联解包→格式化→打包 |
| `format_thesis.py` | 正文格式化（段落分类 + OOXML 属性写入 + 变更报告 + 图表编号检查） |
| `format_headers_footers.py` | 页眉页脚生成（奇偶页、分节页码） |

## 覆盖的格式规范

| 批注 | 内容 |
|------|------|
| 0 | 页面边距 A4, 页眉页脚 |
| 1-4 | 封面（论文类型/题目/个人信息/日期） |
| 6-9 | 独创性声明、授权声明 |
| 10-15 | 中英文摘要、关键词 |
| 16 | 目录标题及一二三级目录项 |
| 17 | 符号说明 |
| 18-20 | 一二级标题、正文 |
| 21 | 脚注 |
| 22-23, 27-28 | 图表标题、图表注释 |
| 25, 31 | 三四级标题 |
| 26 | 公式段落 + 分章自动编号 |
| 33 | 参考文献 + 悬挂缩进 |
| 34-38 | 致谢、附录、个人简历 |
