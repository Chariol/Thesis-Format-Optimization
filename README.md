# BUPT 学位论文格式化工具

自动格式化北京邮电大学硕/博士学位论文（`.docx`），严格遵循《模板-理工科（2025年1月15日）》中全部 39 条批注规范。

## 功能

- **正文格式**：一至四级标题、正文、摘要、关键词、目录、图表题注、公式、脚注、参考文献、致谢等段落的字体/字号/行距/缩进/大纲级别
- **公式自动编号**：分章编号 `(X-Y)`，右对齐，居中显示
- **页眉页脚**：奇数页"本章标题"（STYLEREF 域）、偶数页"北京邮电大学硕士学位论文"；前置部分罗马页码、正文阿拉伯页码
- **页面设置**：A4、上下 2.54cm、左右 3.17cm
- **空行清理**：删除表格/图片前后多余空行

## 使用

```bash
# 依赖
pip3 install lxml

# 一键格式化
bash format.sh 你的论文.docx
```

输出格式化后的 `.docx`，原文件自动备份为 `*_backup.docx`。

## 文件说明

| 文件 | 作用 |
|------|------|
| `format.sh` | 入口脚本，串联解包→格式化→打包 |
| `format_thesis.py` | 正文格式化（段落分类 + OOXML 属性写入） |
| `format_headers_footers.py` | 页眉页脚生成（奇偶页、分节页码） |
